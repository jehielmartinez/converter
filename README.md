##Converter

Converter es un convertidor de masa en las unidades de KG a LBS y viceversa.

##Instalacion

```npm install converter```

##Uso
``` 
import converter from 'converter'

converter(5,"lbs") // 2.27 kg
converter(5, "kg") // 11.02 lbs
```
